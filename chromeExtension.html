In order to interact with this website (as well as others), we have built a Chrome Extension supplement. The extension allows you to see any website via one of the three types of colorblindness. Unfortunately, we do not have a developers account, so in order to use the extension, please follow the below instructions:
  1. Download the extension here. Unzip the file.
  2. Open up Chrome's Preferences (in the menu option, Chrome -> Preferences)
  3. Click Extensions on the left menu bar
  4. Check Developer mode on the top right hand side
  5. Click Load Unpacked Extension... and select the folder the was downloaded/unzipped.
  6. A mini Pokemon should appear now next to the URL bar! Click on the Sandslash to start using the extension when on a website.
  7. When you click the Sandslash, a dropdown menu should apepar in the upper right hand corner. Simply choose the color blindness type you'd like to see through, and it will immediately morph the page for you!
