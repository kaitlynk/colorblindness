The Chrome Extension works as follows:
  1. All DOM elements on the current page are selected.
  2. As each element is traversed through, both the CSS styling for background-color and font color are obtained.
  3. Since the colors are in RGB format, they must be changed to XYZ.
  4. Once in XYZ form, the colors are tested to see if either is on one of the color confusion lines (given by the selected option in the dropdown menu).
  5. If so, that node element is pushed onto an array for that respective color confusion line. This is because we will need to replace each color in the array with the same one color, since color confusion lines represent all colors that cannot be distinguished between.
  6. Otherwise, the node element gets pushed onto an array containing all other colors.
  7. After all elements are traversed through, we now go through each color confusion line array.
  8. We set a line that does not go through the confusion line, and calculate the intersection between that line and the confusion line. This gives us a point to use to replcae all colors within that confusion line.
  9. We inversely calculate the XYZ coordinate to RGB, and replace each node element with the new color.
  10. Taking each element in the array of all colors, we now need to see if they would be colors that could not be seen because of color blindness, but do not exist on a color confusion line.
  11. If so, we replace the unseen colors as necessary.
